# Apex Merchant Analytics | E-Commerce Performance Dashboard
# Overview

Apex Merchant Analytics is a business intelligence dashboard built to analyze and monitor e-commerce sales performance across India. The solution provides actionable insights into revenue trends, regional performance, product categories, and payment behaviors through interactive visualizations.

# Objective

To design and implement a scalable analytics dashboard that enables stakeholders to:

Track sales and revenue performance in near real time

Evaluate regional market penetration

Analyze product category contributions

Support data-driven business decisions

# Tech Stack

Data Visualization: Power BI

Data Transformation & ETL: Power Query

Features: Custom slicers, interactive filters, multi-page drill-downs

# Data Source

E-Commerce Transactional Dataset

Sales transactions

Customer locations across India

Product categories (Clothing, Electronics, Furniture)

Payment methods and transaction details
